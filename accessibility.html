Accessibility Report

Accessibility in web design ensures that websites are usable by everyone, including people with disabilities. It promotes inclusivity, improves user experience, and expands the audience by removing barriers to access. By following accessibility principles, we create digital spaces that are not only equitable but also more functional and user-friendly for all.

Short report on accessibility

As a group, we focused on universal design principles to create an accessible and user-friendly website. We used semantic HTML5 elements, such as nav, main, section, header, and footer, to provide a clear structure that benefits screen readers and enhances the websiteâ€™s usability. To ensure logical organization, we collaboratively maintained a proper heading hierarchy (h1-h4), making the content easy to navigate for all users.

We implemented details and summary elements for expandable content sections, allowing users to reveal information as needed without overwhelming the page layout. Our navigation menu is consistent across all pages, with descriptive link text that avoids generic terms like "click here," ensuring clarity and accessibility. External links open in new tabs (target="_blank) with clear descriptions, helping users understand the context of their actions.

For visual design, we worked together to maintain high contrast ratios, ensuring readability for users with visual impairments. We also used CSS variables to standardize styles and make the design easier to maintain. To improve accessibility on all devices, we implemented responsive design techniques, ensuring the site adapts to various screen sizes. Interactive elements include hover states, providing visual feedback that enhances usability.

Regarding images and media, we ensured all images included descriptive alt text or empty alt attributes for decorative elements, supporting screen reader accessibility. Images were optimized to scale properly and maintain their aspect ratios to prevent distortion across devices. These collaborative coding choices reflect our shared commitment to creating an inclusive, functional, and aesthetically pleasing website for diverse users.
